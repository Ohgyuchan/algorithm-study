## 2.1 fork
#### 2.1.1 git clone과 fork의 차이

> 
**clone** : github에 있는 **내 레포지토리**를 로컬로 복제
**fork** : github에 있는 **다른 사람의 레포지토리**를 복제

즉, fork는 오픈소스와 같이 **다른 사람의 레포지토리**를 가져와 **수정하고, 기능을 추가**할 때 사용하는 기능을 말합니다.

이렇게 fork한 내 레포지토리를 forked repository라고 부릅니다.
